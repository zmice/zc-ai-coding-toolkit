## 概述
对代码进行安全审计和加固，识别漏洞并提供修复方案。

## 审计清单
- [ ] 输入验证和清洗
- [ ] SQL 注入防护
- [ ] XSS 防护
- [ ] CSRF 防护
- [ ] 认证和授权检查
- [ ] 敏感数据处理（加密、脱敏）
- [ ] 依赖漏洞扫描
- [ ] 错误信息泄露
- [ ] 速率限制和防暴力破解
- [ ] 日志中的敏感信息

## 输出格式
```
🔴 高危: [漏洞描述] → [修复方案]
🟡 中危: [漏洞描述] → [修复方案]
🔵 低危: [漏洞描述] → [修复方案]
```

## 使用方式
指定要审计的文件或模块，我将进行安全分析。
