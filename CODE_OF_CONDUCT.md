# Code of Conduct

This project adopts the Contributor Covenant.

## Our standards

Examples of behavior that helps build a healthy community:

- be respectful and specific
- criticize ideas and implementations, not people
- prefer evidence, examples, and reproducible reports
- keep scope and collaboration boundaries clear

Examples of unacceptable behavior:

- harassment, threats, or personal attacks
- discriminatory language
- bad-faith arguments or deliberate disruption
- publishing private information without consent

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, code, or other contributions that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, open a private report through the repository owner’s contact channel or GitHub profile.
