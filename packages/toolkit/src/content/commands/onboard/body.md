调用 codebase-onboarding 技能。系统化探索陌生代码库，建立全局认知。

1. 扫描目录结构 — 识别项目类型、构建系统、关键配置
2. 定位入口点 — 主入口文件、路由定义、数据模型
3. 分析依赖 — 核心三方库、内部模块划分、数据流向
4. 识别架构模式 — 整体架构、状态管理、错误处理、测试策略
5. 生成项目概览 — 技术栈、目录结构、入口点、核心模块、开发命令

## 使用方式

在 `/onboard` 后告诉我你想了解的项目或代码库，我将系统化地探索并生成概览。

### 示例

```
# 探索当前项目
/onboard 帮我了解这个项目的整体架构和核心模块

# 了解特定模块
/onboard 我需要了解支付模块的实现，重点是回调流程和状态机

# 新人入职
/onboard 我是新加入的开发者，帮我快速了解项目的技术栈和开发规范
```
